
# Adult Income Classification (Census Income)

## a) Problem statement
Predict whether an individual's annual income exceeds **$50K** based on demographic and employment attributes. This is a **binary classification** task with target `income` having classes `<=50K` and `>50K`.

## b) Dataset description
- **Source:** Adult (also known as Census Income), UCI Machine Learning Repository; Kaggle mirrors also exist.
- **Instances:** ~48,842; **Features:** 14 predictors (+ target). Mixed categorical & numeric.
- **Target:** `income` (`<=50K`, `>50K`).
